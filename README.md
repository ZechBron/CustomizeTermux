# Customize Termux

You can customize your termux using this very simple script

## Installation:

> pkg install git -y

> git clone https://github.com/ZechBron/CustomizeTermux

> cd CustomizeTermux


## Setup:
 `It is recommenede to run the setup.sh first`

> chmod +x setup.sh

> ./setup.sh

## To Run:

> ./TermuxStyle
